스프링 부트용 템플린 엔진인 thymeleaf를 이용하는 방법
- html 페이지를 반환하는 웹 서버를 개발할 때마다 템플릿 엔진이 하는 일을 알고 빌드하는 방법을 알 수 있음

1. adding the first template and controller
object 색인 생성 완료 첫번 째 템플릿을 추가 -> 소스 메인 내부에 배치해야한다. 리소스 템플릿

-- index.html

2. html파일을 프로젝트 컨트롤러에도 연결해야하므로 메인 코틀린 파일에서 kt 클래스를 만든 후 컨트롤러를 작성한다.
메인 함수에는 @Controller라는 주석이 필요하다.

3. return "index" -> templates 폴더 내부에서 index라는 파일 이름
를 이용해서 브라우저에서 해당하는 템플릿을 렌더링 할 수 있다.

4. 변수의 값을 넘기고 싶을 때는 model을 이용해서 템플릿에 직접 넣을 수 있다. 인수를 model로 받고
model.addAttribute("name", "vlad")를 이용해서 name이라는 변수에 vlad의 값을 넘겨 템플릿에 넘기면
html 상위 옵션에 xmlns:th="http://www.thyemleaf.org"의 값을 추가하고
단락에 th으로 전달 후 text에 만들었던 변수 이름을 이용해서 화면에 출력할 수 있다.

5. Interation in Thymeleaf attributes
리스트 값을 테이블로 보내어 화면에출력하는 방법이다. 4번 과 같지만 데이터 클래스 방법을 이용하여 추가한다.

th=each -> each는 요소가 여러 개일 때 각각에 대하여 반복하여 함수를 실행

6. adding css to thymeleaf
static폴더에 css파일을 생성한다.
html에서 설정하는 방법은
링크 태그를 이용하고
th:href="@{main.css}" rel="stylesheet"
@를 사용하는 형식을 이용해 css 파일을 입력한다.
